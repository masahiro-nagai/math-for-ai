AIの仕組みを支える数理を、計算の途中式まで追いながら理解するための学習アプリです。 本の章立てに合わせて、検索・推薦・画像分類・Transformer・フーリエ解析の核心を 小さな実験として触れられるようにしました。

このアプリでできること

  • 式の入力を変えて、結果がどう動くかを見る
  • 途中計算を追いながら、理論を視覚的に理解する
  • 本の読書メモを残し、自分の言葉で整理する

日本語は簡易トークナイズです。検索系の実験では、単語の間をスペースで区切るとより分かりやすく試せます。

Interactive Labs

式を動かす

TF-IDF ラボ

文書集合とクエリから TF, DF, IDF, スコアを計算します。

計算結果

TF-IDF(d, q) = Σ tf(t, d) × idf(t)

協調フィルタリング ラボ

ターゲットユーザーと近い嗜好を持つ人を探し、未評価アイテムを予測します。

計算結果

similarity(u, v) = (u · v) / (||u|| ||v||)

CNN と Softmax ラボ

小さな画像行列とカーネルを使って、畳み込みと確率的予測を追います。

計算結果

softmax(zᵢ) = exp(zᵢ) / Σ exp(zⱼ)

Attention ラボ

1つのトークンが、どのトークンをどれだけ参照するかを数値で見ます。

計算結果

Attention(Q, K, V) = softmax(QKᵀ / √dₖ)V

離散フーリエ変換 ラボ

時間領域の信号を周波数領域へ写し替え、支配的な周波数を探します。

計算結果

X(k) = Σ x(n) e-i2πkn/N
