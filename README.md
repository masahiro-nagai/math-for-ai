# Math for AI Playground

AIの仕組みを支える数学を、手を動かしながら理解するためのシングルページ学習アプリです。

## できること

- `TF-IDF` を計算して検索順位を観察する
- `コサイン類似度` と中心化を使って推薦を試す
- `畳み込み` と `Softmax` と `交差エントロピー` を小さな行列で追う
- `Attention` の重み計算を `Q/K/V` 行列から確認する
- `離散フーリエ変換` で時間領域と周波数領域を見比べる

## 使い方

`index.html` をブラウザで開くだけで動きます。ビルドは不要です。

## GitHub Pages

このリポジトリは `main` ブランチへの push をきっかけに GitHub Pages へ自動デプロイされます。

- 公開 URL: `https://masahiro-nagai.github.io/math-for-ai/`
- 反映タイミング: `main` に push してから数十秒から数分
- デプロイ設定: `.github/workflows/deploy-pages.yml`

初回だけ GitHub 側で Pages のデプロイ準備に少し時間がかかることがあります。

## ねらい

書籍を読み進めるときに、

- 式の各項が何を意味しているか
- 入力を変えると出力がどう変わるか
- 複数の章にまたがって共通する考え方は何か

を自分の手で確かめられるようにしています。
